# 🎬 Simple Video Landing Page

This is a simple responsive landing page that plays a full-screen background video.

## 🚀 Features
- Full screen video background
- Auto play, loop, muted support
- Mobile and desktop friendly
- Clean and minimal design

## 📁 Files
- index.html (main page)
- video.mp4 (background video file)

## ▶️ How to Run
1. Download or clone this repository
2. Make sure `video.mp4` is in the same folder as `index.html`
3. Open `index.html` in any modern browser (Chrome recommended)

## 🌐 Live Preview (optional)
If hosted on GitHub Pages:
https://your-username.github.io/your-repo-name/

## ⚠️ Note
If video does not play, check:
- File name is exactly `video.mp4`
- Browser supports HTML5 video
- Video format is MP4 (H.264 recommended)
